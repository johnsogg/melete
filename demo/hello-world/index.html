Hello World Graphics Demo

This is the first demo of the Melete graphics library. It demonstrates the new DrawingSurface architecture with layer-based rendering and model-driven graphics. Click the canvas to toggle the circle on and off!

This demo showcases:

  • DrawingSurface with layer schema pattern
  • Model-driven rendering with reactive updates
  • Layer-based drawing with onDemand callbacks
  • Complete semantic drawing API: drawRect, drawCircle, drawText, drawLine
  • Intersection types for combining geometry and styling (DrawRectParams & DrawingStyle)
  • Persistent styling with setStyle() for text properties
  • Comprehensive styling options: fill, stroke, colors, stroke thickness, fonts
  • Semantic clear() method with background color support
  • Mouse event handling with simplified event objects

The Melete library aims to make graphics programming accessible while maintaining flexibility for complex visualizations. This simple example demonstrates the library's straightforward API for creating visual content programmatically.
