Melete Graphics Library Demos

Welcome to the Melete graphics library demo collection. Each demo showcases different features and capabilities of the library for creating web-based visual content.

  • Hello World

    A basic introduction to the Melete library featuring simple shapes, colors, and text rendering on a canvas.

  • Layers Demo

    Demonstrates the layer system with a space-themed scene featuring multiple static layers: starry background, spaceship, and asteroids.

  • Animation Demo

    Shows animated layers in action with twinkling stars and drifting asteroids. Demonstrates the time-based animation system with smooth 60 FPS rendering.

More demos will be added as the library develops...
